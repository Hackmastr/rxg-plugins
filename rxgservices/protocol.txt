Command Listing
-------------------------------------------------------------------------------
HELLO [name] [game]

  Identify the server that has connected. This command must be issued
  after a game server connects to be able to issue requests.

  [name] The name of the server connecting.
  [game] The game the server is running. 
         Possible values: "csgo", "tf", "mc"
